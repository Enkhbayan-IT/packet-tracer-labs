# Uppgift 2 - Prototyping a Network

## 🎯 Objective
Build a simple local area network (LAN) and verify connectivity between hosts.

---

## 🧪 Lab Tasks
- Create a topology with:
  - 2 PCs
  - 1 Switch
- Connect the devices correctly
- Configure IP addresses manually
- Verify connectivity using `ping`

---

## 🛠️ Skills Practiced
- Basic LAN design
- Switch-based topology
- Manual IP addressing
- Connectivity testing with ping

---

## ✅ Result
The network was successfully configured and both PCs were able to communicate with each other.
