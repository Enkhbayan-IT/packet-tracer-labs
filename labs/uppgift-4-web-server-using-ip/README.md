# Uppgift 4 - Connecting to a Web Server Using IP

## 🎯 Objective
Verify connectivity to a web server and access it through a browser using its IP address.

---

## 🧪 Lab Tasks
- Test connectivity to the web server using `ping`
- Open the web browser on a client PC
- Access the server using its IP address
- Observe any first-connection delay caused by ARP resolution or session setup

---

## 🛠️ Skills Practiced
- Server connectivity testing
- Web access validation
- Browser testing in Packet Tracer
- Understanding ARP delay on first connection

---

## ✅ Result
The web server was reachable and the client successfully connected using the server’s IP address.
