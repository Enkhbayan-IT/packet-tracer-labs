# Uppgift 7 - Troubleshooting a Wireless Connection

## 🎯 Objective
Troubleshoot wireless connectivity issues by checking addressing, DNS, and wireless settings.

---

## 🧪 Lab Tasks
- Investigate failed access to a website / URL
- Check if the issue is DNS-related
- Review whether the client is using static IP or DHCP
- Verify wireless settings:
  - SSID
  - Security settings (for example WEP)
- Review Linksys / wireless device configuration
- Restore successful connectivity

---

## 🛠️ Skills Practiced
- Wireless troubleshooting
- DNS troubleshooting
- DHCP validation
- Client configuration checks
- End-to-end connectivity testing

---

## ✅ Result
The root cause of the wireless connectivity issue was identified and corrected, restoring successful access.
