# Uppgift 6 - Using the Ipconfig Command

## 🎯 Objective
Use the `ipconfig` command to inspect host network settings and troubleshoot configuration issues.

---

## 🧪 Lab Tasks
- Review the IP configuration of the client device
- Identify incorrect network settings
- Check the default gateway
- Correct the configuration issue
- Verify connectivity after the fix

---

## 🛠️ Skills Practiced
- `ipconfig` usage
- Host-side troubleshooting
- Default gateway validation
- IP configuration correction
- Connectivity verification

---

## ✅ Result
The incorrect configuration was identified and corrected, restoring successful communication.
