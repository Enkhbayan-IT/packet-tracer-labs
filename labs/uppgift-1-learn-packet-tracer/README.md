# Uppgift 1 - Learn to Use Packet Tracer

## 🎯 Objective
Learn the basics of Cisco Packet Tracer by creating a simple topology and observing how devices communicate.

---

## 🧪 Lab Tasks
- Create a topology with:
  - 2 PCs
  - 1 Hub
- Connect devices using the correct cable type
- Configure IP addresses manually
- Test connectivity using `ping`
- Observe ARP and ICMP packets in Simulation mode
- Review the ARP table

---

## 🛠️ Skills Practiced
- Basic topology creation
- Manual IP configuration
- ARP behavior
- ICMP / Ping testing
- Packet flow observation
- Simulation mode basics

---

## ✅ Result
The devices were successfully connected and communication was verified using ping.  
ARP and ICMP packet flow was observed in Packet Tracer simulation mode.
