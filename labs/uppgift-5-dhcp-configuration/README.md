# Uppgift 5 - DHCP Configuration

## 🎯 Objective
Configure DHCP on a multi-function network device and verify automatic IP assignment for clients.

---

## 🧪 Lab Tasks
- Create a topology with:
  - 3 PCs
  - 1 Linksys / multi-function device
- Change the default router IP address
- Configure the DHCP scope
- Set client devices to obtain IP addresses automatically
- Verify assigned IP addresses and connectivity

---

## 🛠️ Skills Practiced
- DHCP basics
- Automatic IP assignment
- GUI-based device configuration
- Client IP verification
- End-to-end connectivity testing

---

## ✅ Result
Clients successfully received IP addresses from DHCP and network communication was verified.
