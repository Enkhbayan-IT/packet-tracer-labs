# Uppgift 3 - Observing Packets Across the Network

## 🎯 Objective
Observe how packets travel across the network and use `tracert` to understand packet paths.

---

## 🧪 Lab Tasks
- Verify network connectivity
- Use `ping` to test communication
- Use `tracert` to identify the packet path
- Observe packet movement in Simulation mode
- Identify where packets are lost when routers are involved

---

## 🛠️ Skills Practiced
- ICMP troubleshooting
- `tracert` command usage
- Packet path analysis
- Simulation mode analysis
- Basic multi-hop troubleshooting

---

## ✅ Result
Packet flow was successfully analyzed and `tracert` was used to understand the route taken through the network.
