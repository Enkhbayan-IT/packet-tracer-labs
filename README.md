# 📡 Packet Tracer Labs

This repository contains my Cisco Packet Tracer networking labs and troubleshooting exercises.

## 🧠 Skills Demonstrated
- Basic network topology creation
- IP addressing and subnetting
- ARP and ICMP analysis
- Ping and tracert troubleshooting
- Web server connectivity testing
- DHCP configuration
- DNS troubleshooting
- Wireless network troubleshooting

## 🧪 Labs Included
1. Learn to use Packet Tracer
2. Prototyping a Network
3. Observing Packets Across the Network
4. Connecting to a Web Server Using IP
5. Configuring DHCP on a Multi-function Device
6. Using the Ipconfig Command
7. Troubleshooting a Wireless Connection
